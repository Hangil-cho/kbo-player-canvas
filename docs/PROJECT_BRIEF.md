# 프로젝트 브리프

## 프로젝트 이름

KBO Player Lab

## 한 줄 설명

KBO 선수의 기록을 단순히 보여주는 것을 넘어, 그 기록이 만들어진 이유와 변화 과정을 해설하는 선수 분석 웹페이지입니다.

## 핵심 컨셉

사용자가 선수를 선택하면 다음 내용을 한 페이지에서 확인할 수 있습니다.

1. 선수 기본 정보와 핵심 성적
2. 기본 스탯과 상세 스탯
3. 리그 및 포지션 내 상대 위치
4. 성적 변화의 원인 분석
5. 연도별, 월별 변천사
6. 표, 그래프, 사진, 영상 기반 설명

## 1차 대상

처음에는 타자 1명 분석 페이지부터 만듭니다.

타자부터 시작하는 이유:

- 타율, 출루율, 장타율, OPS처럼 직관적인 지표가 많습니다.
- 리그 평균 비교와 변화 분석을 설명하기 쉽습니다.
- 투수의 구종, 회전수, 움직임 데이터보다 데이터 확보 난이도가 낮습니다.

## 1차 MVP 범위

1차 버전에서는 다음만 구현합니다.

- 선수 선택
- 시즌 핵심 지표 카드
- 리그 평균 대비 비교
- 연도별 OPS 변화
- 월별 타격 흐름
- 성적 변화 원인 해설
- 기본 기록표

## 이후 확장 범위

2차 이후에는 다음을 추가합니다.

- 투수 분석
- 수비/포수 분석
- Statiz 기반 세이버 지표
- 선수별 사진 영역
- 참고 영상 링크
- 기사/인터뷰 기반 변화 스토리
- 구종/코스/타구 질 분석
- 자동 리포트 생성

## 중요한 원칙

- 초보자도 이해할 수 있게 보여줍니다.
- 지표 이름만 나열하지 않고, 왜 중요한지 설명합니다.
- 데이터 출처와 계산 방식을 문서화합니다.
- 확실하지 않은 해석은 확실하지 않다고 표시합니다.
- 공개 데이터 이용 가능 범위와 사이트 정책을 존중합니다.
